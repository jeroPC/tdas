# Seguimiento Paso a Paso: Longest Increasing Subsequence (LIS)

**Vector a evaluar:** `[3, 1, 4, 5]`  
**Estado Inicial:** `DP = [1, 1, 1, 1]` *(Cada número por sí solo mide 1)*

---

## 1. Simulación del Algoritmo

### Paso 1: `i = 0` (Valor actual: **3**)
* **Bucle `j`:** No tiene elementos a la izquierda.
* **Estado de DP:** `[1, 1, 1, 1]`

### Paso 2: `i = 1` (Valor actual: **1**)
* **Comparación:** `j = 0` (valor 3) $\rightarrow$ ¿$1 > 3$? **No**.
* **Estado de DP:** `[1, 1, 1, 1]`

### Paso 3: `i = 2` (Valor actual: **4**)
* **Comparación 1:** `j = 0` (valor 3) $\rightarrow$ ¿$4 > 3$? **Sí**.  
  * `DP[2] = max(1, DP[0] + 1)` $\rightarrow$ `max(1, 1 + 1) = 2` *(Secuencia: [3, 4])*
* **Comparación 2:** `j = 1` (valor 1) $\rightarrow$ ¿$4 > 1$? **Sí**.  
  * `DP[2] = max(2, DP[1] + 1)` $\rightarrow$ `max(2, 1 + 1) = 2` *(Secuencia: [1, 4])*
* **Estado de DP:** `[1, 1, 2, 1]`

### Paso 4: `i = 3` (Valor actual: **5**)
* **Comparación 1:** `j = 0` (valor 3) $\rightarrow$ ¿$5 > 3$? **Sí**.  
  * `DP[3] = max(1, DP[0] + 1)` $\rightarrow$ `max(1, 1 + 1) = 2`
* **Comparación 2:** `j = 1` (valor 1) $\rightarrow$ ¿$5 > 1$? **Sí**.  
  * `DP[3] = max(2, DP[1] + 1)` $\rightarrow$ `max(2, 1 + 1) = 2`
* **Comparación 3:** `j = 2` (valor 4) $\rightarrow$ ¿$5 > 4$? **Sí**.  
  * `DP[3] = max(2, DP[2] + 1)` $\rightarrow$ `max(2, 2 + 1) = 3` *(Secuencia: [3, 4, 5] o [1, 4, 5])*
* **Estado de DP:** `[1, 1, 2, 3]`

---

## 2. Matriz de Transición de Estados

| Índice `i` | Valor Vector | Comparaciones (`j`) | Decisión / Condición | Estado DP Resultante |
| :---: | :---: | :--- | :--- | :---: |
| **0** | **3** | Ninguna | Caso base | `[1, 1, 1, 1]` |
| **1** | **1** | `j=0` (v:3) | $1 > 3$ es Falso. No cambia. | `[1, 1, 1, 1]` |
| **2** | **4** | `j=0` (v:3), `j=1` (v:1) | $4 > 3 \rightarrow$ Usa $DP[0]+1$. | `[1, 1, 2, 1]` |
| **3** | **5** | `j=0` (v:3), `j=1` (v:1), `j=2` (v:4) | $5 > 4 \rightarrow$ Usa el acumulado $DP[2]+1$. | `[1, 1, 2, 3]` |

---

## 3. Resultado Final

El valor máximo dentro del arreglo `DP` es **3**.  
La longitud de la subsecuencia creciente más larga es **3**.
